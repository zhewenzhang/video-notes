📊 AI 进展的三个 Scaling Laws

1️⃣ 预训练扩展定律 (Pre-training)

  • Gemini 3 证明了预训练扩展定律依然有效
  • 没人知道为什么有效，只是经验观察
  • 像"古埃及人对太阳的理解"——能精确测量，但不完全理解原理

2️⃣ 后训练扩展定律 (Post-training / RLVR)

  • Reinforcement Learning with Verified Rewards
  • 任何可验证的事情都可以自动化
  • "Reasoning saved AI" —— 填补了 18 个月的硬件空白

3️⃣ 测试时计算扩展 (Test-time Compute)

  • 让模型在推理时"思考更久"
  • ARC-AGI 从 8% → 95% 只用了 3 个月
  • 新的进步维度，不依赖预训练规模

🔧 GPU vs TPU：Nvidia vs Google

维度Nvidia (GPU)Google (TPU)
策略全栈解决方案只做 TPU + 其他网络芯片
优势生态系统完整成本优势
产品Hopper → BlackwellTPU v6 (2024) → v7 (2025)
部署难度高（液冷、3000 磅机架）低（更成熟）

Blackwell 的挑战

  • 冷却：风冷 → 液冷
  • 机架重量：~1000 磅 → ~3000 磅
  • 功耗：30 kW → 130 kW
"Imagine if to get a new iPhone you had to change all the outlets in your house to 220 volt, put in a Tesla power wall, put in a generator, put in solar panels..."

🚀 XAI 的优势

为什么 XAI 会第一个发布 Blackwell 模型？

  • 建设速度 - Jensen Huang: "没人比 Elon 更快建数据中心"
  • 合作关系 - XAI 帮 Nvidia 调试 Blackwell
  • 优势循环 - 最快部署 → 最早发现问题 → 帮所有人解决

💡 关键数据

  • 最大相干 GPU 集群：200,000 Hoppers
  • Blackwell 部署时间：2025 年末开始规模化
  • Blackwell 模型预测：2026 年初，来自 XAI
  • ARC-AGI 进步：8% → 95% (3 个月，reasoning)
