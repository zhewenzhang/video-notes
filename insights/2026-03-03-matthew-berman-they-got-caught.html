核心事件

Anthropic 公开指控三家中国 AI 实验室

被指控的公司：
  • DeepSeek (深度求索)
  • Moonshot AI (月之暗面)
  • MiniMax (名之梦)
指控内容：
  • 通过 Anthropic API 进行工业化蒸馏攻击 (Distillation Attack)
  • 超过 1600 万次 Claude 交互
  • 系统性提取 Claude 的能力来训练和改进自家模型
---

什么是蒸馏攻击？

蒸馏 (Distillation) - 合法方法

  • 将大模型的知识"蒸馏"到更小的模型中
  • 优点：更小、更快、更高效
  • 缺点：损失一些质量和保真度
  • 正常用途：创建可本地运行的高效模型

蒸馏攻击 (Distillation Attack) - 滥用方法

  • 通过 API 大规模查询对手模型
  • 用查询结果训练竞争性模型
  • 本质上是在"偷取"对手的能力
  • 违反了 API 使用条款

关键背景

被指控公司的地位

  • 这三家公司正在生产全球最好的开源模型
  • DeepSeek 以高性价比模型闻名
  • Moonshot (月之暗面) 是中国领先的 AI 初创

为什么现在曝光？

  • AI 竞争白热化

  • API 监控技术成熟，能检测异常使用模式
